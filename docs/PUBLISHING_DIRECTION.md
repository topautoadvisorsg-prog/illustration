# Publishing Direction — Illustration Density & Page Design

**This is guidance, not a hard rule or a fixed formula.** It is the publishing
direction we currently believe produces the strongest books while staying
practical and scalable. The publishing layer should apply professional judgment,
not mechanical enforcement.

Where it lives in code: `WILDLANDS_PUBLISHING_STYLE` in
`backend/src/services/page-quality/page-quality-review.ts` (the advisory
Page Quality Review director). Its `principles`, `visualPresenceGoal`, and
`illustrationLayers` encode this document.

## Core philosophy

Not an art book. Not a textbook. A **premium illustrated wilderness publication**.
The reader should feel every page was intentionally designed.

## Visual presence on every page

Almost every page should carry **some** visual element — this does NOT mean every
page needs a major illustration. It can be a full illustration, a partial/side/
corner illustration, a botanical study, a field-guide element, a map fragment, a
decorative naturalist accent (pine branch, specimen study), or wilderness-themed
framing. The goal: pages never feel visually abandoned. Premium books feel
designed even when text-heavy.

## The three illustration layers

1. **Feature Art** — chapter openers, major landscapes, signature wildlife, hero
   artwork. Visually impressive landmark pages.
2. **Supporting Illustration** — 25% / 50% layouts, corner/side layouts, reference
   and educational layouts. Support learning and visual rhythm.
3. **Visual Identity** — botanical accents, small naturalist sketches, decorative
   wilderness elements, page framing, ornaments. Maintain visual quality without
   major artwork.

> Layer 3 maps to the `repeatable` layouts (Text-Heavy, Back-Matter, Margin-
> Illustration): a shared decorative accent generated once and reused across the
> book (see the repeating shared asset feature).

## Page-rhythm strategy (a pattern we are considering, not a rule)

Illustration page + text page (or illustration spread + text spread) is a strong
default rhythm: strong visual impact, easier production/review/automation, cleaner
reading. **It must not become repetitive.** Keep using mixed, feature, comparison,
reference, illustration-dominant, and text-dominant layouts as appropriate.

## Controlled variety, not uniformity

Keep using the full layout set — full-page, 50/50, 25%, corner, side, comparison,
reference. Objective: controlled variety, not sameness.

## Color interior

The strategy assumes color interiors, so maximize their value — a color page
should feel like a color page. This does not mean forcing large illustrations
everywhere; it means every page makes meaningful use of visual design.

## Publishing-agent responsibility

Evaluate manuscript content, chapter purpose, page density, illustration
opportunities, visual rhythm, and readability — then decide how to distribute
visual elements across the chapter.

## Avoid mechanical rules

Do NOT blindly enforce "one image every X pages", fixed percentages, or rigid
patterns. Use these ideas as guidance.

## Final goal

The reader should feel they are holding a premium wilderness encyclopedia, field
guide, or expedition volume. Some pages art-heavy, some text-heavy — all
intentionally designed and professionally published.

## Current alignment (as of this writing)

The Page Quality Review already embodies much of this: Feature / Mixed / Text-first
bands, target percentages, and advisory findings for rhythm, layout repetition,
feature balance, awkward continuations, and underfilled pages — all advisory, never
mechanical. This document is now its source of truth; the profile's `principles`,
`visualPresenceGoal`, and `illustrationLayers` reflect it.
